# TyranoScriptの開発環境テンプレート

> [!IMPORTANT]
> TyranoScriptのライセンスに即して、このテンプレートにティラノスクリプト本体は一切同梱していない。

## Setup

```sh
# setup TyranoScript development environment
git clone https://github.com/Myxogastria0808/tyrano-template.git
cd tyrano-template
sh setup.sh
npm i

# start vscode
cd src
code .
```

## Usage

```sh
# run this command in the root directory of this repository
npm run dev
```
